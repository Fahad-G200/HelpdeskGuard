# Brukerveiledning – HelpdeskGuard

## Hvordan bruke appen
1. Aapne appen.
2. Registrer bruker.
3. Logg inn.
4. Gaa til "Saker".
5. Fyll inn tittel og beskrivelse.
6. Velg kategori og prioritet.
7. Trykk "Send inn sak".

## Hva appen gjor i v1.0
- Lar bruker registrere konto og logge inn.
- Lar bruker opprette en ny sak via skjema.
- Har navigasjon mellom Hjem, Saker, Konto og Informasjon.

## Feilsoking
| Problem | Mulig arsaksforklaring | Losning |
|---|---|---|
| Faar ikke logget inn | Feil e-post/passord | Kontroller inndata og prov igjen |
| Registrering feiler | E-post finnes allerede | Bruk annen e-post eller logg inn |
| Saken vises ikke i liste | Saksliste er ikke ferdig i v1.0 | Funksjonen er planlagt |

## Personvern
Data lagres lokalt i appen i denne prototypen.
Det brukes ikke ekstern skytjeneste i v1.0.
