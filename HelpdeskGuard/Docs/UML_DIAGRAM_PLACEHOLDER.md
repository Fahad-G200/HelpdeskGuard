# UML Diagram Placeholder

Dette er en plassholder for UML-diagrammer.

Plan:
- Lag klassediagram for sentrale typer (AuthStore, Ticket, TicketStore).
- Lag sekvensdiagram for registrering/innlogging og opprettelse av sak.

Verktøy:
- draw.io (diagrams.net) eller tilsvarende.

Når diagrammet er klart:
1. Eksporter som PNG/SVG/PDF.
2. Legg filen i `Docs/`.
3. Oppdater denne filen med lenke til endelig diagram.
